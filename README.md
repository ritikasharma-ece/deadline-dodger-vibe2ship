# deadline-dodger-vibe2ship
A tool for stressed students facing absolute deadlines. It converts chaotic syllabus copies, notes, or project briefs into an actionable, hour-by-hour study roadmap, generates high-density micro cheat-sheets, and builds quick stateful quizzes to check your retention. Built with Python, Streamlit, and the Gemini 1.5 Flash SDK for the Hackathon.

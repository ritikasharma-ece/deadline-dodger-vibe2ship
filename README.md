# Deadline Dodger

Deadline Dodger is a study assistant built for the Vibe2Ship Hackathon.

The idea came from a simple problem: when deadlines pile up, it's difficult to figure out where to start. Instead of spending time planning, this app helps students get an organized study plan in minutes.

Just provide your syllabus, notes, or project details, and the app generates a study roadmap, short revision notes, and a few quiz questions to help you revise.

## Features

- Creates a study plan based on your input
- Generates concise revision notes
- Creates practice quiz questions
- Simple and easy-to-use interface
- Powered by Google Gemini

## Built With

- Python
- Streamlit
- Google Gemini API

## Running the Project

Clone the repository:

```bash
git clone https://github.com/ritikasharma-ece/deadline-dodger-vibe2ship.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Start the application:

```bash
streamlit run app.py
```

## Why I Built This

As a student, I know how stressful it gets when multiple deadlines are close together. I wanted to build something that could save time by organizing study material into a practical plan instead of making students do everything manually.

This project was built as part of the Vibe2Ship Hackathon using Python, Streamlit, and Google Gemini.
